🚀 빠른 시작 (5분)

  1. 병동명 설정 → 상단 ⚙️ 규칙설정 → "병동 표시" 섹션에서 본인 병동명 입력
  2. 간호사 등록 → 우측 👥 패널 → 데모 간호사 8명 제거 후 실제 인원 추가
  3. 특수 속성 설정 → 임산부·야간전담·프리셉터 등 카드에서 토글
  4. 자동배치 실행 → 좌상단 ✦ 자동배치 클릭 → 결과 확인
  5. 충돌 확인 및 수정 → ⚠️ 경고 → 셀 클릭으로 수정
  6. 저장 → 💾 저장 (자동저장도 30초마다 작동)
  7. Excel 출력 → ↓ Excel 또는 ⎙ 인쇄

👥 간호사 관리

등록: 우측 👥 패널 → 하단 입력란에 이름·직급·입사일 입력 → ＋ 추가

수정: 카드 클릭으로 펼치기 → 정보 변경 (자동 저장)

삭제: 카드 펼치기 → 🗑 삭제 (확인 후)

순서 변경: 카드 드래그 → 원하는 위치에 놓기

직급 5종

  • 수간호사: 평일 D 고정(설정 가능). 자동배치에서 제외
  • 주임/차지: 선임. 각 시프트에 최소 N명 배치 (규칙설정 가능)
  • 평간호사: 일반. 각 시프트 최소 N명 배치
  • 신규간호사: 야간 단독 배치 금지, 월 야간 한도 등 보호 규칙 적용

선호도

간호사를 클릭하면 현재 설정된 정보가 요약 표시됩니다(활성 항목만). ✎ 편집을 누르면 이름·직급·특수속성·교대 선호도(D/E/N 배지 클릭: 중립→선호→기피)·파트너·사진·색상 등 전체 항목을 수정할 수 있습니다. 접힌 카드에서도 임산부🤰·고정근무 등은 아이콘으로 요약 표시됩니다.

⭐ 특수 속성

간호사 카드의 "특수 속성" 영역에서 설정. 자동배치 시 반영됩니다.

  • 🤰 임산부: 야간 금지 + 임산부 2명 이상 동일 듀티 회피
  • 🌙 야간전담: 야간만 배치 (D/E 금지). 월 한도 14회
  • 🚫 야간면제: 건강상 사유 등으로 야간 X
  • 고정 근무: Day/Eve/Night 중 하나만 배정 (사이클 근무자)
  • 🎓 프리셉티 지정: 주임·차지·평간호사 카드에서 신규간호사 선택 → 같은 듀티 우선 배치
ⓘ 한 신규간호사는 한 명의 프리셉터만 가질 수 있습니다 (자동 1:1 보장).

🔁 반복 근무 패턴

특정 간호사가 일정한 패턴으로 반복 근무할 때 사용합니다 (예: 야간전담의 NNNOO 반복, 고정 사이클 근무자).

설정 방법 — 간호사 카드를 펼치면 특수 속성 아래 🔁 반복 패턴 입력란이 있습니다.

  • 패턴을 듀티 코드로 입력: 예) NNNOO = 야간 3일 + 오프 2일 반복
  • 사용 가능한 글자: D · E · N · O · V · B
  • 시작일 = 이 패턴이 며칠째부터 시작될지 (1 = 1일부터)

동작 — 자동배치를 실행하면 패턴 간호사는 시작일 기준으로 패턴이 월 전체에 순환 적용되고, 나머지 간호사가 그 위에서 규칙에 맞게 채워집니다.

ⓘ 이미 입력한 연차(V)·경조(B)와 원티드(고정 신청)는 패턴이 덮어쓰지 않고 그대로 보존됩니다.

시프트 종류 (5가지):

  • D Day · 07:00-15:00
  • E Eve · 15:00-23:00
  • N Night · 23:00-07:00
  • O Off (통합)
  • V 연차 (잔여에서 차감)
  • B 경조/공가 등 특별휴가 (off 정량과 별개)

입력 방법

  1. 시프트바에서 원하는 코드 선택 (예: D)
  2. 캘린더 셀 클릭 → 즉시 배정 (동일 코드 다시 클릭 시 제거)
  3. "해제" 칩 선택 후 셀 클릭 = 상세 모달 열기

드래그 이동

배정된 셀의 D/E/N 아이콘을 드래그하여 다른 셀로 이동. 둘이 자동 교환됩니다.

편집 락 🔒

시프트바 좌측 🔓 버튼 클릭 → 조회 전용 모드. 자동배치·수정 모두 차단

✦ 자동배치

실행: 상단 ✦ 자동배치 버튼 → 확인 → 약 1초 후 완료

🧩 미리 배치 + 나머지만 자동 채우기

수간호사가 일부 인원을 미리 손으로 배치해 두고, 빈 칸만 자동으로 채울 수 있습니다. 칸을 채워둔 상태에서 자동배치를 누르면 "기존 배치 유지" 선택이 나타납니다.

  • 체크 ON(기본): 미리 채운 칸은 그대로 두고 빈 칸만 자동 배치
  • 체크 OFF: 원티드(🔒)만 남기고 전체를 새로 배치

※ 원티드 락(🔒)은 이 옵션과 무관하게 항상 보존됩니다. (개인 신청 확정분과 수간호사 임의 배치는 별개)

동작 원리

  • 가중치 기반 확률 선택 알고리듬
  • 30가지 전략 중 무작위 선택 → 실행할 때마다 다른 결과 (마음에 안 들면 재실행)
  • 완료 토스트에 사용된 전략명 표시 (예: "전략: 회전 +5/+3")

반영되는 규칙

  • 인력 구성 (선임/평간호사 비율)
  • 연속 근무 한도, 야간 후 휴식
  • 금지 패턴 (NOD/EOD 등)
  • 간호사 속성 (임산부/야간전담/프리셉터)
  • 원티드 락 (100% 보존)
💡 같은 달에 자동배치를 여러 번 실행해서 가장 마음에 드는 결과를 선택할 수 있습니다.

🔋 피로도 점수

간호사별 한 달 근무의 피로 정도를 점수로 보여주는 평가 지표입니다. 간호사 이름 옆 색점(●) + 숫자로 표시되고, 클릭하면 상세 내역이 열립니다.

⚠️ 피로도는 보기용 평가 지표일 뿐, 자동배치 결과에는 전혀 영향을 주지 않습니다. "누가 이번 달 많이 힘든가"를 한눈에 보기 위한 참고 정보입니다.

근거 — HSE Fatigue Risk Index

영국 보건안전청(HSE)의 Fatigue Risk Index 방법론을 근무표 데이터에 맞게 단순화했습니다. 세 가지 요소를 합산합니다:

  • ① 근무 타이밍: 야간(N)이 일주기 리듬을 가장 교란 → 높은 가중치 (기본 N=8·E=3·D=1)
  • ② 누적: 연속 근무가 길수록 + 단일 off(하루만 쉼) + 야간 연속
  • ③ 회전 방향: 역회전(N→D 등 거꾸로 도는 근무)에 페널티

색 구간: 초록 양호 · 주황 보통 · 빨강 과로

ⓘ 모든 가중치와 색 구간은 규칙설정 → 🔋 피로도 점수 섹션에서 현장에 맞게 조정할 수 있습니다.

⚙️ 규칙설정

상단 ⚙️ 버튼 → 규칙설정 모달 열기. 8개 섹션:

  1. 병동 표시: 병동명·설명
  2. 인력 구성: 시프트별 최소 선임/평간호사 수
  3. 연속 근무 한도: 최대 야간 연속·근무 연속·야간 후 휴식
  4. 수간호사: 평일 시프트·주말/공휴일 Off
  5. 신규간호사: 야간 단독 금지·월 야간 한도·연속 야간 금지
  6. 🚫 금지 패턴: NOD/EOD/NNNN 등 8가지 켜기/끄기
  7. 🌙 야간 세부 규칙: 연속 야간 / 월 야간 한도
  8. 👤 속성 규칙: 임산부·야간전담·프리셉터 페어링

🛌 off 이월 · 경조 휴무

off(휴무)는 "그달에 꼭 다 써야 하는 정량"이 아니라 다음 달로 이월되는 누적 잔고로 관리됩니다. 못 쓴 off는 사라지지 않습니다.

off 목표 계산

  • 기본 = 그달의 휴일 수 (토 + 일 + 공휴일)
  • 이월 = 지난 달까지 못 쓴(또는 초과한) off 누적분
  • 목표 = 기본 + 이월

off 잔고 현황판

간호사 관리 패널 상단 🛌 off 현황 버튼 → 전원의 기본·이월·목표를 한눈에 봅니다. 간호사 카드의 "Off N/M" 옆에 * 표시가 있으면 이월분이 반영된 것 (마우스 올리면 분해 표시).

월마감 (이월 누적)

  1. 한 달 듀티를 다 짠 뒤, off 현황판에서 📋 이번 달 마감 클릭
  2. 이번 달 못 쓴 off가 각자 이월 잔고에 더해짐 (초과 사용분은 차감)
  3. 다음 달 목표에 그 이월분이 자동 반영됨
⚠️ 월마감은 월말에 한 번만 누르세요. 같은 달 두 번 누르면 이월이 중복됩니다 (경고가 뜹니다).

off 목표 수동 조정

특정 간호사의 off 목표를 직접 정하려면, 간호사 카드 → "🛌 off 목표 조정"에 숫자 입력. 비우면 자동(기본+이월)으로 돌아갑니다.

B 경조/공가 휴무

결혼·장례 등 경조사나 공가는 연차(V)·off(O)와 별개인 특별휴가입니다. 시프트 팔레트에서 경조(B)를 선택해 배정하세요. B는 off 목표 계산에 포함되지 않습니다 (법정 유급 특별휴가이므로).

① 수간호사 열람용 — 전체 근무표 (A4 한 장)

모든 간호사 × 한 달 전체를 A4 가로 한 장에 압축한 격자표입니다. 근무 관리·게시용으로 적합합니다.

  • 각 칸에 듀티 코드(D·E·N·O·V·B)만 표시 — 색 + 야간(N) 굵게로 흑백 프린터에서도 구분
  • 이름 옆 연-숫자 = 남은 연차 (연차 소진 관리용)
  • 주말·공휴일 칸 음영 + 공휴일 첫 글자 표시 (예: 어린이날 → "어")
  • 출력 옵션(체크): 개인별 합계(D/E/N/Off) · 날짜별 인원(Day/Eve/Night, 최소인원 미달 빨강) · 듀티 색상(흑백 프린터는 끄기 권장)

② 개인 간호사별 — 월간 탁상 달력

간호사 한 명씩 한 페이지(A4 세로) 달력 형식입니다. 개인에게 나눠주는 용도로 적합합니다.

  • 날짜 칸마다 본인 근무를 큼직하게 (Day/Eve/Night, 쉬는 날은 OFF 강조)
  • 같은조 = 그날 같은 듀티 동료 (성으로 표시)
  • 전/후 시프트 담당자 = 인수인계 상대. 인계 흐름 Day→Eve→Night→(익일)Day 기준 —
    · Day 근무: 전 = 전날 Night 담당자 / 후 = 당일 Eve 담당자
    · Night 근무: 전 = 당일 Eve 담당자 / 후 = 다음날 Day 담당자
    인계 상대의 성향을 미리 알면 인계 시점 조율에 도움이 됩니다.
  • 대상: 전원(간호사마다 한 페이지) 또는 선택한 한 명

인쇄는 별도 창에서 열리며, 브라우저 인쇄 다이얼로그에서 종이 출력 또는 PDF 저장을 선택할 수 있습니다. 색이 안 나오면 인쇄 설정에서 "배경 그래픽" 옵션을 켜세요.

❔ 자주 묻는 질문

Q. 다른 사용자가 내 데이터를 볼 수 있나요?

A. 절대 아닙니다. 모든 데이터는 본인 브라우저에만 저장됩니다. 같은 URL이라도 다른 사람과는 완전히 격리됩니다.

Q. 공용 PC를 여러 명이 쓰는데 괜찮나요?

A. 같은 윈도우 계정에서 같은 브라우저를 쓰면 데이터가 공유됩니다. 별도 크롬 프로필을 만들거나, 시크릿 모드를 사용하거나, 사용 후 백업하고 데이터를 삭제하세요.

Q. 자동배치 결과가 마음에 안 듭니다.

A. ✦ 자동배치를 다시 클릭하면 다른 전략이 적용됩니다 (총 30가지). 또는 ↩ 되돌리기로 직전 상태로 복원 가능.

Q. 브라우저를 바꾸면 데이터가 사라지나요?

A. 네, localStorage는 브라우저별로 분리되어 있습니다. 파일저장(JSON) → 새 브라우저에서 파일열기 → 복원하세요.

Q. 한 셀에 두 가지 코드를 넣을 수 있나요?

A. 한 셀에 한 코드만 가능합니다. 휴가(V)와 오프(O)는 별개입니다.

Q. 자동배치가 너무 느립니다.

A. 간호사 20명 이하는 1초 이내. 50명 이상은 2~3초 걸릴 수 있습니다. 잠시 기다려주세요.

Q. 충돌이 너무 많이 나옵니다.

A. ⚙️ 규칙설정에서 일부 규칙을 끄거나 한도를 완화해보세요. 또는 인력이 부족한 경우 인력 구성 최소치를 낮추세요.

Q. 데이터를 영구적으로 안전하게 보관하려면?

A. 매월 ↑ 파일저장으로 JSON 백업 → 클라우드(구글 드라이브 등)에 보관 권장.

🛌 off 잔고 현황판
off는 정량이 아니라 누적 잔고입니다. 이번 달 못 쓴 off는 소멸되지 않고 다음 달로 이월됩니다.
· 기본 = 그달 휴일 수(토+일+공휴일) · 이월 = 지난 달까지 누적분 · 목표 = 기본+이월
⚠️ 월마감은 이번 달 미소진 off를 각자 이월잔고에 더합니다. 듀티를 다 짠 뒤 월말에 한 번만 누르세요. (중복 실행 시 경고)
🛠️ 이월잔고 직접 수정
각 간호사의 이월잔고를 직접 입력해 바로잡습니다. (양수=다음 달로 이월, 음수=초과사용 차감)
잘못 누적된 값이 있으면 올바른 값을 적거나, 아래 전원 0 초기화로 처음부터 다시 시작하세요.
⚠️ 초기화하면 그동안의 마감기록도 함께 지워집니다. 되돌릴 수 없으니 필요하면 먼저 데이터를 내보내(💾) 두세요.
