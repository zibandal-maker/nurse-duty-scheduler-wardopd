📅 공휴일 관리

모든 부서(듀티·외래)가 같은 공휴일·행사 데이터를 공유합니다. 여기서 한 번 갱신·추가하면 전 부서에 반영됩니다.

“행사”는 달력에 표시만 됩니다. “공휴일”은 근무 자동배치에서 제외됩니다(공휴일 갱신이 안 될 때 직접 지정용).

💾 전체 데이터 백업 · 이전

모든 데이터(직원·부서·전 부서 근무표·공휴일·행사·설정)를 파일 하나로 저장합니다. 다른 컴퓨터에서 “복원”하면 이 파일과 똑같은 상태로 재현됩니다 — 시연·이전용. 복원은 그 컴퓨터의 기존 데이터를 전부 덮어씁니다.

📖 사용 가이드 — 처음 오셨나요?

이 프로그램은? 병동 듀티(D·E·N·O 3교대)와 외래(오전·오후) 근무표를 한곳에서 관리하는 도구입니다. 모든 직원은 직원관리에 한 번만 등록하고, 각 부서로 배치합니다.

① 직원 등록 — 먼저 직원관리에서 직원을 등록합니다(이름·생일·연차·사진). 한 직원은 한 부서에 소속되며, 기본 정보는 전 부서가 공유합니다.

② 부서 선택 — 위 카드에서 병동·외래를 고르면 해당 근무표로 들어갑니다. 새 외래 부서는 “＋ 새 외래 부서 개설”로 만듭니다.

③ 근무표 작성 — 병동은 자동배치·규칙 기반 3교대, 외래는 붓으로 칠하거나 자동배치(평일 근무·토요일 오전)로 만듭니다.

④ 한눈에 보기 — 오늘 현황판은 그날 부서별 근무자를, 월 통계는 부서별 인원·근무일·연차를 모아 보여줍니다.

⑤ 공휴일·행사 — 위 “공휴일 관리”에서 한 번 갱신·추가하면 전 부서에 반영됩니다.

⑥ 다른 PC로 시연·이전 — “전체 백업”으로 파일을 저장해 옮기고, 그 PC에서 “복원”하면 똑같이 재현됩니다.

ⓘ 정보 · 면책 · 데이터 처리

간호사 근무표 자동 배치 도구 — 브라우저 단독 실행, 서버 불필요.

⚠️ 면책 조항 — 본 도구는 데모·실험 목적으로 제공됩니다. 실제 의료 현장에서의 안전성·정확성·법적 요건(근로기준법·의료법 등) 충족 여부는 검증되지 않았으며, 사용 결과에 대해 제작자는 책임지지 않습니다. 실제 운영 전 자체 검증과 법규 확인이 필요합니다.

🔒 데이터 처리 — 모든 데이터는 사용자 브라우저(localStorage)에만 저장되며 외부 서버로 전송되지 않습니다. 같은 URL을 다른 사람이 접속해도 본인 데이터는 공유되지 않습니다(브라우저 보안 모델).

⚠️ 공용 PC 주의 — 같은 계정·같은 브라우저를 여러 명이 쓰면 데이터가 서로 보일 수 있습니다. 별도 브라우저 프로필, 또는 사용 후 백업·삭제, 시크릿 모드를 권장합니다.

직원은 한 부서에 소속되며, 이름·생일·연차 등 기본 정보는 전 부서가 공유합니다.
